Software: TEQLY | Cloud

Sie nutzen bereits oder wollen in Zukunft Dropbox, Google Drive, WeTransfer, OneDrive oder einen anderen Cloud-Dienst nutzen? Nehmen Sie Ihre Cloud mit TEQLY selbst in die Hand! Unsere, auf der Open Source Plattform von Nextcloud, basierte Lösung bietet Ihnen absolute Kontrolle und Sicherheit über Ihre Daten. Wir hosten Ihre persönliche Cloud auf einem hochverfügbaren Server in einem schweizer Rechenzentrum. Sie haben zu jederzeit volle Kontrolle wer auf Ihre Daten Zugriff hat.
Holen Sie sich die Kontrolle über Ihre Daten zurück: Daten die auf Servern in den USA oder China landen, gibt es mit der TEQLY | Cloud nicht.

Sicher, stabil, schweizerisch

Entdecken Sie die vielseitigen Vorteile einer eigenen Cloud:

  • Collaboration Space

    Kommunikation ist das A und O in einem Unternehmen. Mit der "Talk"-Funktion haben Sie die Möglichkeit auf Ihrem Cloud Server Text-, Sprach- und Videochats mit Mitarbeitern und Kunden abzuhalten. Da die ganze Kommunikation über Ihren eigenen Server läuft, fallen Sicherheis- und Datenschutzbedenken weg. In der TEQLY | Cloud können Sie, Ihre Mitarbeiter und Ihre Kunden einfach und sicher kommunizieren.

  • Online Office

    Das mobile, digitale Büro auf einer Plattform? Die "Office"-Funktion macht es möglich Ihre Daten, den Kalender, den Mail Client, Ihre Kontakte und weitere Elemente auf einer Plattform zu bündeln. Sie können von überall auf der Welt auf Ihr digitales Büro zugreifen und orts- und zeitunabhängig arbeiten. Das ganze findet auf Ihrem Server statt, was bedeuted, dass alle Daten zu jederzeit unter Ihrer Kontrolle sind und bleiben.

  • Anpassungsfähig

    Strahlen Sie mit einer gebrandeten Cloud Oberfläche professionalität aus. Mittels einer Subdomain (cloud.ihredomain.ch) kann das System paralell zu Ihrer bereits bestehenden Website gehostet werden. Alternativ bieten wir Ihnen auch eine Subdomain von uns an. Wir passen die Öberfläche gerne mit den Farben und Logos gemäss Ihrer Corporate Identity an. Interne wie externe User nehmen Ihre Marke oder Ihr Unternehmen so auch bei der Nutzung Ihres Cloud Services wahr.
